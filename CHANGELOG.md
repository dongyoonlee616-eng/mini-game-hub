# CHANGELOG

Mini Game Hub의 버전별 변경 사항을 기록합니다.

## [1.1.0] - 2026-06-29

### Added

- 싱글 플레이 게임 6개 추가
  - 색깔 기억하기
  - 10초 맞추기
  - 빠른 계산
  - 폭탄 피하기
  - 목표 숫자 만들기
  - 순간 판단

- 멀티 플레이 게임 8개 추가
  - 색깔 기억하기 대결
  - 순발력 대결
  - 버튼 함정 대결
  - 계산 대결
  - 영역 점령전
  - 같은 그림 찾기 대결
  - 실시간 해전
  - 다트 대결

- 메인 메뉴에 싱글 / 멀티 플레이 필터 추가
- 메인 메뉴 게임 카드에 게임 방법 모달 추가
- 각 게임별 게임 방법 설명 추가
- 모바일 중심의 2인용 양면 게임 화면 추가
- 일부 게임에 최고 기록 저장 기능 추가
- 터치 중심 조작에 맞춘 모바일 UI 개선

### Changed

- 게임 번호 체계를 싱글 플레이와 멀티 플레이 기준으로 분리
- 메인 메뉴의 PLAY 버튼과 게임 방법 버튼 위치 및 크기 조정
- 일부 게임의 버튼 배치, 점수 표시, 안내 문구 개선
- 멀티 게임 일부를 한 기기에서 위아래로 플레이하기 좋게 화면 회전 구조로 변경
- 빠른 터치 / 조이스틱 / 캔버스 기반 게임을 위한 모바일 조작 구조 개선

### Fixed

- 모바일 브라우저에서 CSS / JS 캐시가 남아 변경 사항이 바로 보이지 않던 문제 대응
- 일부 게임에서 버튼 연타로 중복 입력되던 문제 수정
- 실시간 해전에서 명중 시 턴 유지 및 공격 잠금 처리 추가
- 일부 모바일 화면에서 게임판 또는 버튼이 잘리는 문제 수정

## [1.0.0] - 2026-06-28

### Added

- Mini Game Hub 프로젝트 초기 정식 버전 완성
- 모바일 중심 미니게임 허브 메인 페이지 추가
- 밝은 흰색 계열 배경과 검정 계열 포인트 컬러 기반 디자인 적용
- 싱글 플레이 / 멀티 플레이 게임 필터 기능 추가
- `games-data.js` 기반 게임 카드 자동 생성 구조 추가
- GitHub Pages 배포 설정 추가

### Games

- 반응속도 테스트 게임 추가
  - 화면 변화 후 터치 반응속도 측정
  - 최고 기록 저장 기능 추가

- 클릭 속도 테스트 게임 추가
  - 3초 카운트다운 후 5초 동안 터치 수 측정
  - CPS 계산 기능 추가
  - 최고 CPS 저장 기능 추가

- 숫자 야구 게임 추가
  - 3자리 숫자 추리
  - Strike / Ball 판정
  - 시도 기록 표시

- 카드 뒤집기 게임 추가
  - 싱글 플레이 기억력 게임
  - 4×4, 5×5, 6×6 게임판 선택 기능
  - 5×5 보너스 카드 처리
  - 시도 횟수, 맞춘 짝, 플레이 시간 표시

- 양면 탁구 게임 추가
  - 한 기기에서 위아래 2명이 플레이하는 멀티 플레이 게임
  - 터치 기반 패들 이동
  - 먼저 5점 달성 시 승리
  - 느림 / 빠름 공 속도 선택 기능
  - 라운드 시작 전 2초 카운트다운 적용
  - 득점 후 2초 뒤 다음 라운드 시작

- 턴제 카드 뒤집기 게임 추가
  - 2인용 턴제 기억력 게임
  - 같은 카드를 맞추면 현재 플레이어 점수 획득
  - 정답 시 턴 유지
  - 오답 시 상대 턴으로 변경
  - 최종 점수에 따라 승리 / 무승부 판정

### Changed

- 게임 HTML 파일 구조 정리
  - 기존 게임별 폴더 구조에서 `public/games/` 내부 HTML 파일 구조로 변경

- CSS 파일 구조 분리
  - 공통 스타일은 `style.css`
  - 게임별 스타일은 각 게임 전용 CSS 파일로 분리

- 메인 메뉴 구조 개선
  - 게임 카드를 HTML에 직접 작성하지 않고 JavaScript 데이터 기반으로 자동 렌더링하도록 변경

- 데스크톱 안내 문구 위치 조정
  - 모바일 미리보기 화면을 가리지 않도록 오른쪽 안내 박스 형태로 수정

- 양면 탁구 게임 진행 방식 개선
  - START 버튼 클릭 후 바로 시작하지 않고 2초 카운트다운 후 시작하도록 변경
  - 득점 후 바로 다음 라운드가 시작되지 않고 2초 뒤 시작되도록 변경

### Technical

- 데이터베이스 없이 동작하는 정적 웹사이트 구조 유지
- GitHub Actions를 통한 GitHub Pages 배포 설정
- 모바일 브라우저 캐시 문제 대응을 위해 CSS/JS 버전 쿼리 사용
- 게임별 파일 분리로 유지보수성 개선
- `games-data.js`를 통해 게임 정보 관리 구조 개선

### Notes

- 이번 버전은 MGH의 첫 정식 배포 버전입니다.
- 이후 버전에서는 새로운 미니게임을 계속 추가하는 방식으로 확장할 예정입니다.
